# java第一次实战案例——图书管理系统

## 1 概述
### 1-1.需求分析
 图书管理系统需实现的功能如下：
（1）用户管理：包括用户的注册于登录。
（2）图书管理：包括录入、查询、修改和删除图书信息。
（3）借书：包括借阅图书和查看借书记录。
（4）还书：包括还书和查看还书记录。
（5）为了保证系统安全，进入系统时，对用户登录的密码进行加密与解密。
